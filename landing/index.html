VarejoSync

Sistema de gestão de varejo desenvolvido como prova de conceito para o Projeto Integrador.

Funcionalidades

• Dashboard gerencial

• Controle de estoque

• Sistema de vendas (PDV)

• Indicadores operacionais

Tecnologias

Frontend: HTML, CSS, JavaScript

Backend: Node.js + Express

Banco: SQLite

Ferramentas de Desenvolvimento

VSCode – desenvolvimento do código

Git e GitHub – versionamento do projeto

Postman – testes dos endpoints da API

Projeto Integrador

Curso: Análise e Desenvolvimento de Sistemas

Projeto acadêmico demonstrando integração entre frontend, backend e banco de dados.
